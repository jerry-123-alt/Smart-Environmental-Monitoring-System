# Smart Environmental Monitoring System

## Project Overview

This project is a real-time IoT environmental monitoring system based on ESP32.

The system collects environmental data through multiple sensors, processes the data locally, displays the results on an OLED screen, uploads data via WiFi, stores the data in a cloud database, visualizes the data through a web dashboard, and triggers alarms when environmental values exceed preset thresholds.

---

## Features

- Real-time environmental monitoring
- ESP32 embedded controller
- OLED display
- WiFi communication
- Cloud database
- Web dashboard
- Alarm notification

---

## Hardware

- ESP32 Development Board
- DHT22 Temperature & Humidity Sensor
- BH1750 Light Sensor
- MQ135 Air Quality Sensor
- OLED Display
- Active Buzzer
